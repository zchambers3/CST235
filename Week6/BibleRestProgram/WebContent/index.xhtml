Bible Rest Service

Bible Rest Services

Works for first three books. Genesis, Exodus, and Leviticus

Get Verse

Search by the name of a bible book, chapter number, and verse number. It returns a Bible object that contains the book name, chapter number, verse number and scripture.

http://localhost:8080/BibleRestProgram/rest/bible/getverse/{book}/{chapter#}/{verse#}

Get First Occurrence of word

Search for the first time a word appears in the bible. It returns a Bible object containing the book name, chapter number, verse number, and the scripture.

http://localhost:8080/BibleRestProgram/rest/bible/findfirstword/{searchword}

Get word count

Search the bible for a word and returns the number of times it occurs in the bible.

http://localhost:8080/BibleRestProgram/rest/bible/getwordcount/{searchword}
